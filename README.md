## 프로젝트 소개
자산 관리 및 금융에 대한 지식이 부족한 저년층 대상으로 편리한 금융 정보 전달을 목표로 하는 프로젝트입니다.
## 개발 기간
22.09.22 ~ 22.05.03
## 구성 멤버
- 박지은 
   - 유저 정보 관리용 서버 구축 및 관리
   - 유저 정보 관리위한 DataBase 생성 및 관리와 이를 위한 API 개발
   - 안드로이드와 AI 서버 통신을 위한 API 개발
